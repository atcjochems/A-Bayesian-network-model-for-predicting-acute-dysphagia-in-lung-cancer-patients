User manual

1. Download the files from this repository by clicking the 'Download ZIP' button
2. Install Hughin researcher, available at http://www.hugin.com/
3. Double click Model.net
3. Hugin opens and the Bayesian network can be observed

For an elaborate guide to using Bayesian network models, please see http://download.hugin.com/webdocs/manuals/Htmlhelp/
Click Manual, Run mode for an explanation on how to run the model and make predictions.
